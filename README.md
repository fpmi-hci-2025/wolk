Project Name: **YazichOK — платформа оценки устной речи**

Description: Кроссплатформенное решение (iOS + Web) для экспресс-оценки английской речи. Пользователь записывает ответ, система расшифровывает аудио (ASR), анализирует текст, выдаёт уровень CEFR (A1–C2), подсвечивает лексику, грамматические ошибки и даёт рекомендации по перефразированию. Поддерживаются отчёты, история прогресса и геймификация.

Key features:
- Устный тест: выбор темы, запись/загрузка аудио, отправка на сервер.
- Автоматический анализ: ASR → CEFR-классификация, грамматика, словарь, рекомендации.
- Отчёты и прогресс: история результатов, визуализация, экспорт PDF/Excel.
- Доп. модули: AI-чат, словарь пользователя, чтение/аудирование, геймификация.
- Клиенты: iOS (Swift) и web (Flutter, адаптивный UI); сервер на Go.

Architecture:
- Backend: Go, API Gateway с JWT/OAuth, очереди (RabbitMQ/Kafka), хранилище аудио (S3), Postgres для данных, Elasticsearch для поиска, экспортный сервис (PDF/Excel), аналитика (Prometheus/Grafana).
- External: ASR-провайдер (Google Speech-to-Text или аналог), интеграция с LMS/SSO.
- Clients: запись/кеш офлайн, отправка в API, отображение отчётов и прогресса.
